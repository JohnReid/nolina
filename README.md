## Nolina

Naive NumPy (and eventually TensorFlow) implementations of randomised linear algorithms, including:

| Method             | NumPy         | TensorFlow |
| ------------------ |:-------------:| ----------:|
| power              | [x]           | [ ]        |
| inverse iteration  | [x]           | [ ]        |
| Rayleigh iteration | [x]           | [ ]        |
| Jacobi             | [ ]           | [ ]        |


### Installation

```bash
python setup.py install
```


### Development

```bash
conda env create -f environment.yml
```
